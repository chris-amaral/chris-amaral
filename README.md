## 🔐 Segurança da Informação

Atuação consolidada em **DevSecOps e Security Engineering** em ambientes regulados de **fintech de delivery e bancos Tier 1 da América Latina** — com requisitos rigorosos de disponibilidade, confidencialidade, integridade e conformidade.

### 🎯 Áreas de domínio

| Domínio | Atuação prática |
| --- | --- |
| 🛡️ **Application & API Security** | WAF (AWS WAF, Cloudflare), API Gateway hardening, **Rate Limiting** contra abusos e invasões em ambiente transacional de alto volume — **-40% em custos operacionais** com bloqueio de tráfego malicioso |
| 🔑 **Identity & Access Management** | IAM granular em AWS/Azure/GCP/OCI, **Least Privilege** por padrão, RBAC em Kubernetes, federação SSO/OIDC, MFA e princípios de **Zero Trust** |
| 🗝️ **Secret & Key Management** | HashiCorp Vault, AWS Secrets Manager, Sealed Secrets, External Secrets Operator — rotação automatizada e zero-segredo-em-repositório |
| 🚨 **Incident Response & Self-Healing** | Sistema de **Self-Healing** em produção (-20% trabalho manual, MTTR otimizado), runbooks automatizados, alertas proativos e detecção via observabilidade (Datadog, Graylog, Splunk) |
| 🧪 **Shift-Left Security (Pipelines)** | **SAST** (SonarQube), **SCA** (Snyk, Trivy, Dependabot), **DAST**, container scanning, IaC scanning (tfsec, Checkov) integrados a GitHub Actions / Jenkins / GitLab CI |
| 📜 **Compliance & Governança** | Adequação a **PCI-DSS** (meios de pagamento), **LGPD**, **ISO 27001** e diretrizes **BACEN** — controles de auditoria, segregação de ambientes, criptografia TLS/at-rest e trilhas de log forenses |
| 🔒 **Network & Cloud Security** | Security Groups, NACLs, VPC peering, Private Endpoints, mTLS via Istio service mesh, Cloudflare Edge Protection, DDoS mitigation |
| 🐳 **Container & Kubernetes Security** | Pod Security Standards, Network Policies, OPA/Gatekeeper, image signing, distroless/minimal images, runtime security |
